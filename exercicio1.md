># Algotitmo para trocar pneu furado

```
 Algoritmo para trocar pneu furado:

- Tirar o triangulo;
- Colocar o triangulo na pista;
- Tirar a chave de roda;
- Afrouxar os parafusos, um a um,  com a chave de roda; ⇒ for parafuso in roda: afrouxa(parafuso);
- Tirar o macaco;
- Levantar o carro até altura desejada;
- Retirar todos os parafusos;
- Retirar a roda;
- Colocar o stepe;
- Acochar os parafusos no stepe;
- Baixar o macaco;
- Reforçar os parafusos novamente;
- Guardar o pneu furado;
- Guardar o macaco;
- Guardar a chave de roda;
- Guardar o trangulo;

FIM DO ALGORITMO
```
