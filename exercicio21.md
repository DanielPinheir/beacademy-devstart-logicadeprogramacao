> ## Validação do usuário

```
Algoritmo "ValidacaoUsuario"
// Lógica de Programação
// Professor   : Marcos Monteiro
// Descrição   : Validação de usuário e senha com o enquanto
// Autor(a)    : Daniel Pinheiro

Var
   usuario: caractere
   senha: caractere

Inicio
   escreval("Digite seu usuario: ")
   leia(usuario)

   enquanto (usuario <> "daniel") faca
            escreval("Usuário inválido!Digite seu usuario novamente: ")
            leia(usuario)
    fimenquanto

   escreval("Digite sua senha: ")
   leia (senha)

   enquanto (senha <> "paylivre") faca
            escreval("Senha inválida! Digite sua senha novamente: ")
            leia(senha)
   fimenquanto
   escreval("Seja muito bem-vindo(a) a Paylivre!!!")
Fimalgoritmo

```
